# Camada de Negócios (Business) - Organizador Pro

Esta pasta contém a documentação da camada de **Negócios (Business)** do projeto, conforme a metodologia BMAD.

## Finalidade

O objetivo desta etapa é alinhar as diretrizes de valor do negócio, compreender as necessidades do usuário, delimitar o escopo macro do sistema e definir as diretrizes operacionais de governança, interface de usuário (UI) e segurança. Isso assegura que o software resolva problemas reais sem introduzir riscos aos dados físicos do usuário ou expor o sistema operacional Windows 11 a vulnerabilidades.
